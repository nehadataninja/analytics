Customer Churn Prediction

  • LinkedIn

Our project focuses on predicting customer churn in e-commerce using machine learning. We analyzed the E-commerce Customer Churn Analysis and Prediction dataset from Kaggle to identify key churn factors like tenure time, complaints, order categories, and payment methods . To handle class imbalance, we applied RandomOverSampler and optimized model performance using GridSearchCV with Stratified K-Fold Cross-Validation . Among the models tested, XGBoost performed best , achieving a ROC-AUC of 0.99 and an F1-score of 92.83 . These insights help businesses implement targeted retention strategies, improve customer engagement, and reduce churn. GitHub repository.
